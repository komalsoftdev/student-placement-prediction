# 🎓 Student Placement Prediction System

## 📌 Project Overview
This project is a Machine Learning-based web application built using Flask that predicts whether a student will be placed or not based on academic and skill-related inputs.

The model uses a Random Forest Classifier trained on a synthetic student dataset.

---

## 🚀 Features
- User-friendly web interface
- Real-time prediction system
- ML model integration with Flask
- Data preprocessing with scaling
- Clean and responsive UI
- Classification output (Placed / Not Placed)

---

## 🧠 Tech Stack
- Python
- Flask
- Pandas, NumPy
- Scikit-learn
- HTML, CSS

---

## 📊 Model Details
- Algorithm: Random Forest Classifier
- Accuracy: ~63%
- Preprocessing: StandardScaler
- Train-Test Split applied
